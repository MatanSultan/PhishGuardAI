# PhishGuard AI Demo Script (Israel segments)

## 30-second intro
- Problem: most incidents start with people. Local phishing via WhatsApp, email, and invoice changes is rising; managers lack visibility.
- Promise: PhishGuard AI gives a clear Risk Score, local simulations, and manager-ready summaries in Hebrew/English without needing a security team.
- Outcome: know who needs help, which channel is risky, and what to do next.

## 2–3 minute core flow (generic)
1. Landing hero: “Make every employee your safest link.” CTA: Book demo / Start pilot. Set expectations for local scenarios.
2. Admin dashboard:
   - Show the Risk Score card (0–100 + reason bullets).
   - “Manager at a glance” card: biggest risk, who needs follow-up, risky channel, next action.
   - Simulation preview: highlight scenarios that match the segment (WhatsApp, invoices, parent messages).
3. Reports:
   - Plain-language summary + Risk Score recap.
   - Manager-at-a-glance strip (risk area, channel, refresher groups, next action).
4. Training/employee view:
   - Personalized summary (supportive, simple).
   - Quick start card with a segment-relevant example.

## Segment-specific mini flows

### Nursing home (2 minutes)
1. Admin dashboard Risk Score: call out supplier invoices, WhatsApp, delivery. Show reasons list.
2. Manager at a glance: weakest area likely “supplier payments” or “delivery”; risky channel WhatsApp.
3. Simulation preview: select WhatsApp supplier change; mention AI summary references supplier/payment pressure.
4. Reports: note refresher group, risky channel, next action “reinforce invoices & WhatsApp approval flow.”

### Education (2 minutes)
1. Risk Score reasons: parent/principal impersonation, Google/Microsoft warnings.
2. Manager at a glance: biggest risk “parent messages”; risky channel email/WhatsApp.
3. Simulation preview: pick parent urgent form + Microsoft reset warning.
4. Reports: refresher group = homeroom teachers; next action = parent-message drill + account warning refresher.

### SMB (90 seconds)
1. Risk Score: invoices, delivery, account alerts.
2. Manager at a glance: show risky channel SMS, follow-up employee name.
3. Simulation preview: courier fee SMS + vendor bank change email.
4. CTA: “Start pilot” → signup; highlight managed add-on if no security staff.

## Key talking points (thread through)
- Local, segment-aware scenarios (email/SMS/WhatsApp) in Hebrew/English.
- Risk Score is explainable: accuracy + inactivity + repeated failures + weak domains + channels.
- Managers get next action, not jargon.
- Works without a security team; managed add-on available.

## What to click (sequence)
1. Landing page CTA → signup/pilot.
2. Admin dashboard: Risk Score card → Manager at a glance → Simulation preview.
3. Admin Reports: plain summary → manager strip → channel breakdown (optional).
4. Training/employee: first-session card → AI personal summary.

## What to say at each step
- Risk Score: “Here’s your 0–100 score and why it’s medium/high—supplier changes + WhatsApp pressure.”
- Manager strip: “One screen: what’s risky, who to re-engage, which channel, and the exact next action.”
- Simulations: “These are local to your segment—parent messages / vendor bank changes / WhatsApp approvals.”
- Reports: “Plain Hebrew/English summary; no SOC needed. Click to see refresher groups.”
- Employee view: “Supportive, simple guidance; no scare tactics.”
